បង្គត់​ចំនួន

នៅ​ក្នុង Math object យើង​អាច​បង្គត់​លេខ ដោយ​ប្រើ method ខាងក្រោម ។

  • round() method បង្គត់​ធម្មតា ។
  • ceil() method បង្គត់​ឡើង​ជានិច្ច ។
  • floor() method បង្គត់​ចុះ​ជានិច្ច ។
  • trunc() method កាត់​លេខ​នៅ​ក្រោយ​ក្បៀស ។
Math.round(7.6)     // តម្លៃ 8
Math.round(7.4)     // តម្លៃ 7
Math.ceil(5.4)      // តម្លៃ 6
Math.floor(3.6)     // តម្លៃ 3
Math.floor(-3.2)    // តម្លៃ 4
Math.trunc(3.6)     // តម្លៃ 3
Math.trunc(3.4)     // តម្លៃ 3
Math.trunc(-3.2)    // តម្លៃ -3

ដឺក្រេ និង​ឫស

  • pow() method ប្រើ​សំរាប់​គណនា​ដឺក្រេ​នៃ​មួយ​ចំនួន ។
  • sqrt() method ប្រើ​សំរាប់​គណនា​ឫស​ការេ​នៃ​មួយ​ចំនួន ។
  • cbrt() method ប្រើ​សំរាប់​គណនា​ឫស​គូប​នៃ​មួយ​ចំនួន ។
Math.pow(7, 3)      // ៧ ស្វ័យ​គុណ​នឹង ៣ ស្មើ​នឹង ៣៤៣
Math.sqrt(49)       // ឫស​ការេ​នៃ ៤៩ ស្មើ​នឹង ៧
Math.sqrt(27)       // ឫស​ការេ​នៃ ២៧ ស្មើ​នឹង ៣

អតិបរមា និង​អប្បបរមា

  • min() method ប្រើសំរាប់​បង្ហាញ​លេខ​តូច​បំផុត ដែល​ត្រូវ​បាន​បញ្ចូល ។
  • max() method ប្រើសំរាប់​បង្ហាញ​លេខ​ធំ​បំផុត ដែល​ត្រូវ​បាន​បញ្ចូល ។
Math.min(-2, 70, -50)       // បង្ហាញ -50
Math.max(-2, 70, -50)       // បង្ហាញ 70

ព្រាវ​លេខ

random() method ប្រើសំរាប់​ព្រាវ​លេខ​ចន្លោះ​ពី ០ ដល់ ១ ។

ត្រីកោណមាត្រ

Math object មាន​ផ្ទុក​នូវ method មួយ​ចំនួន​ដែល​តែង​ឃើញ​ក្នុង​មេរៀន​ត្រីកោណមាត្រ ។ តម្លៃ​ដែល​ត្រូវ​ប្រើ ​គឺ​គិត​ជា​រ៉ាដ្យ៉ង់ ។

  • cos() method បង្ហាញ​តម្លៃ​កូស៊ីនុស​នៃ​អ្វីមួយ ។
  • sin() method បង្ហាញ​តម្លៃ​ស៊ីនុស​នៃ​អ្វីមួយ ។
  • tan() method បង្ហាញ​តម្លៃ​តង់សង់​នៃ​អ្វីមួយ ។
  • acos() method បង្ហាញ​តម្លៃ​អាក់កូស៊ីនុស​នៃ​អ្វី​មួយ ។
  • asin() method បង្ហាញ​តម្លៃ​អាក់ស៊ីនុស​នៃ​អ្វី​មួយ ។
  • atan() method បង្ហាញ​តម្លៃ​អាក់តង់សង់​នៃ​អ្វី​មួយ ។

អ៊ិចស៉្បូណង់ស្យែល និង​លោការីត

  • exp() method គណនា​អយល័រ​ស្វ័យ​គុណ​នឹង​មួយ​ចំនួន ។
  • log() method គណនា​លោការីត​គោល​ធម្មជាតិ​នៃ​មួយ​ចំនួន ។
