មូលដ្ឋាន

សញ្ញាចំណុច​ក្បៀស

ត្រង់​ចំណុច​នេះ Javascript គឺ​ដូច CSS ដែរ ។ រាល់​ដំណើរ​ការ​អ្វី​ចប់​មួយ ត្រូវ​បញ្ចប់​ដោយ​សញ្ញា​ចំណុច​ក្បៀស គឺ​សញ្ញា ; នេះ ។

វិចារណ៍

វិចារណ៍​មិន​បង្ហាញ​អ្វី​ជា​ចម្លើយ​នោះ វា​មាន​ប្រយោជន៍​ដល់​អ្នក​សរសេរ​កូដ ដែល​ប្រើ​សំរាប់​កត់​ចំណាំ​ថា​តើ​កូដ​នៅ​កន្លែង​នោះ និយាយ​អំពីអ្វី ។ នៅ​ក្នុង Javascript ដើម្បី​សរសេរ​វិចារណ៍ សំរាប់​មួយ​បន្ទាត់ ត្រូវ​សរសេរ​ផ្ដើម​ដោយ​សញ្ញា "/" ពីរ​ដង សំរាប់​ច្រើន​បន្ទាត់ ត្រូវ​សរសេរ​ផ្ដើម​ដោយ "/" ហើយ​បញ្ចប់​ដោយ "/" ។

Console

Console គឺ​ជា​កន្លែង​មួយ​ដែល​យើង​អាច​ធ្វើ​តេស្ត​កូដ Javascript ។ ដើម្បី​បើក console យើង​ត្រូវ mouse ស្ដាំ​លើ​ទំព័រ​បច្ចុប្បន្ន រួច​ជ្រើស​យក inspect element ។ វា​នឹង​លោត​កន្លែង​ដែល​យើង​អាច​សរសេរ​កូដ​ក្នុង​នោះ​បាន ហើយ​ពេល​ចុច enter កូដ​ដែល​យើង​សរសេរ​នឹង​ដំណើរការ ។

alert() function

alert() function បង្ហាញ​ប្រអប់​ផុស​ចេញ​មក ។

ឧទាហរណ៍​ខាងលើ

  • onclick attribute សំរាប់​កំណត់​ព្រឹត្តិការណ៍​នៅ​ពេល​ដែល​ចុច​លើ​ធាតុ ។ តម្លៃ​នៃ onclick attribute គឺ​ជា​កូដ Javascript ។
  • នៅ​ក្នុង​វង់​ក្រចក​នៃ alert() គឺ​អនុញ្ញាត​ឲ្យ​អ្នក​សរសេរ​អ្វី​ចូល​ទៅ​ក្នុង ដើម្បី​បង្ហាញ​ក្នុង​ប្រអប់​ដែល​ផុស​ចេញ​មក ។ បើ​សរសេរ​លេខ ត្រូវ​សរសេរ​ធម្មតា ។ បើ​សរសេរ​ពាក្យ​អ្វី​មួយ ត្រូវ​សរសេរ​ក្នុង​អព្ភន្តរសញ្ញា ។

ចាប់​ធាតុ​តាម id

មាន​វិធី​ជាច្រើន​ដើម្បី​ចាប់​យក​អ្វី​មួយ​នៅ​ក្នុង​ទំព័រ​មួយ ។ អ្វី​ដែល​យើង​គួរ​សិក្សា​ក្នុង​មេរៀន​មូលដ្ឋាន​នេះ គឺ​វិធី​ចាប់​យក​ធាតុ​តាម​ឈ្មោះ id នៃ​ធាតុ​នោះ ។

ឧទាហរណ៍​ខាងលើ

  • ឈ្មោះ​នៃ id ត្រូវ​សរសេរ​នៅ​ក្នុង​អព្ភន្តរសញ្ញា​នៅ​ក្នុង​វង់ក្រចក ។
  • .innerHTML ប្រើ​សំរាប់​ចាប់​យក​អ្វីៗ​ដែល​នៅ​ក្នុង​ធាតុ​នោះ ។

ត្រង់​ចំណុច​ទាំងប៉ុន្មាន​នេះ ប្រាកដ​ជា​ពិបាក​សំរាប់​មេរៀន​មូលដ្ឋាន តែ​យើង​នឹង​សិក្សា​ឲ្យ​ស៊ី​ជម្រៅ​នៅមេរៀន​បន្ត​បន្ទាប់​ទៀត ។

រចនា

ដើម្បី​រចនា​ធាតុ អ្នក​ត្រូវ​រចនា​តាម CSS ។ Javascript មាន​លទ្ធភាព​បញ្ជា​ការ​រចនា​ទាំងនោះ​បាន​ផង​ដែរ ។ ដើម្បី​រចនា យើង​ត្រូវ​ចូល​ទៅ​កាន់ style property នៃ​ធាតុ​នោះ ។ រួច​កំណត់​ការ​រចនា​ដោយ​ប្រើ property ដូច property ក្នុង CSS ដែរ ។

ឧទាហរណ៍​ខាងលើ

  1. document.getElementById("a") ប្រើ​សំរាប់​ចាប់​យក​ធាតុ​ដែល​មាន id a
  2. .style សំរាប់​ចូល style property នៃ​ធាតុ​នោះ ។ property នៃ CSS ទាំងអស់​ស្ថិត​នៅ​ក្នុង style property
  3. .background សំរាប់​រចនា​ផ្ទៃ​ខាងក្រោយ​នៃ​ធាតុ (ប្រើ​ពាក្យ​ដូច property នៃ CSS) ។

Property នៃ CSS ដែល​មាន​សញ្ញា - ឧទាហរណ៍ background-image property នៃ CSS សំរាប់​នៅ​ក្នុង Javascript ត្រូវ​សរសេរ backgroundImage ។
