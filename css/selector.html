ចាប់​តាម attribute

ដើម្បី​ចាប់​យក​ធាតុ​ទាំងឡាយ​ទៅ​តាម attribute ណាមួយ ត្រូវ​ប្រើសញ្ញា "[]" ហើយ​ត្រូវ​សរសេរ​លក្ខណៈ​នៃ attribute នោះ នៅ​ក្នុង ។

របៀប​សរសេរ
(ឈ្មោះ attribute តាង​ដោយ attr)		 ឧទាហរណ៍ អត្ថន័យ
[attr] [title] ចាប់​ធាតុ​ដែល​មាន title attribute
[attr = តម្លៃ] [target=_blank] ចាប់​ធាតុ​ដែល​មាន target attribute តម្លៃ _blank
[attr ~= តម្លៃ] [title~=Khode] ចាប់​ធាតុ​ដែល​មាន title attribute តម្លៃមាន​ពាក្យ Khode
[attr |= តម្លៃ] [lang|=km] ចាប់​ធាតុ​ដែល​មាន lang attribute តម្លៃ​ផ្ដើម​ដោយ km
[attr ^= តម្លៃ] [title^="Welcome to"] ចាប់​ធាតុ​ដែល​មាន title attribute តម្លៃ​នាំ​មុខ​ដោយ​ឃ្លា Welcome to
[attr $= តម្លៃ] img[src$=".jpg"] ចាប់ <img> ដែល​មាន src attribute តម្លៃ​បញ្ចប់​ដោយ .jpg
[attr *= តម្លៃ] a[href*=khode] ចាប់ <a> ដែល​មាន href attribute តម្លៃ​មាន​អក្សរ​ khode

តម្លៃ attribute ណា​ដែល​មាន​សញ្ញា​ដក​ឃ្លា ត្រូវ​សរសេរ​តម្លៃ​នោះ​នៅ​ក្នុង​អព្ភន្តរសញ្ញា ។
